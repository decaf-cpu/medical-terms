# 🏥 의학용어 검색

대한의사협회 의학용어집 6판 기반 의학용어 실시간 검색 도구입니다.

## 🔗 바로 사용하기

👉 **[의학용어 검색 시작하기](https://decaf-cpu.github.io/medical-terms/medical_terms_search_standalone.html)**

## 📊 데이터 정보

- **총 50,746개 의학용어** 수록
- 영문 용어 ↔ 한글 뜻풀이 양방향 검색
- 대한의사협회 의학용어집 6판 기반
- 최종 업데이트: 2026년 1월 (clean3 버전)

## ✨ 주요 기능

### 🔍 실시간 검색
- 타이핑할 때마다 즉시 결과 표시
- 300ms 디바운싱으로 부드러운 검색 경험

### 🔄 양방향 검색
- **영문 → 한글**: "fever" 입력 → "열" 검색
- **한글 → 영문**: "복부" 입력 → "abdominal" 관련 용어 검색

### 💡 스마트 검색
- 대소문자 구분 없음 (FEVER = fever = Fever)
- 부분 일치 검색 (abd → abdominal, abdomen 등)
- 검색어 하이라이트 표시

### 📱 반응형 디자인
- 모바일, 태블릿, 데스크톱 모두 지원
- 직관적인 UI/UX

## 🚀 사용 방법

### 온라인 사용
GitHub Pages 링크로 바로 접속하여 사용

### 오프라인 사용
1. `medical_terms_search_standalone.html` 파일 다운로드
2. 파일을 더블클릭하여 브라우저에서 실행
3. 서버 설치 불필요!

## 💻 기술 스택

- **순수 HTML/CSS/JavaScript** (프레임워크 없음)
- **완전히 독립적** (외부 라이브러리 의존성 없음)
- **오프라인 작동** (모든 데이터 포함)

## 📝 검색 예시

```
검색어: "fever"
결과: fever → 열

검색어: "복부"
결과: abdominal → 복부의, abdomen → 복부

검색어: "HEART"
결과: heart → 심장, heartbeat → 심박동

검색어: "17"
결과: 17-ketosteroid → 17-케토스테로이드
```
